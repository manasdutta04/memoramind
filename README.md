# MemoraMind

MemoraMind is a voice companion designed for memory care.

It helps older adults feel calm, recognized, and emotionally supported through natural conversation, while giving families a simple daily view of how their loved one is doing.

## What MemoraMind Delivers

- A warm companion experience that speaks gently and responds with personal context
- A simple setup flow for families to add life memories, routines, and relationships
- A clear family dashboard with mood signals, topics, and distress alerts
- Instant demo mode for quick evaluation

## Product Experience

### 1. Family Setup
Families create a profile once with important personal details:
- name and preferred language
- family relationships
- meaningful life memories
- daily routine and favorite topics

### 2. Companion Conversations
The elder uses one large voice button and talks naturally.
MemoraMind listens, responds with kindness, and keeps replies short and reassuring.

### 3. Family Dashboard
Caregivers get a daily snapshot:
- session summary
- mood state
- key topics
- distress moments and follow-up cues

## Built for Real-World Care

- Designed for clarity, large text, and low-friction interaction
- Supports fallback behavior when external services are unavailable
- Includes a ready-to-use demo profile for rapid judging/testing

## Account & Access

- Sign in and account management are handled through Clerk
- Protected product routes require authentication
- Users add their own provider API keys in **Settings**
